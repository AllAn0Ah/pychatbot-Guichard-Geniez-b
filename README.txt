# pychatbot-Guichard-Geniez-b
project python chatbot Noah guichard et de Gabriel Geniez

Ce fichier a pour but d'informer sur l'utilisation de nos programmes. 
Il est direct et simple pour faciliter la compréhension rapide du projet.

Tout d'abord, pour lancer le programme permettant de répondre aux questions, il faut exécuter le fichier main.py.
Ensuite il faut lire dans la console les instructions données par le menu. 
Puis selon notre choix, il faut sélectionner le nombre adéquat (affiché dans la console) permettant de répondre à la question.
